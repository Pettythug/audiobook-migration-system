# [TICKET-003] Non-Destructive Audio Migration

**Status:** [x] DONE
**Assignee:** Gatekeeper (Merged)

## Goal Description
Build `tests/Migrate-Audiobooks.ps1`. This script must parse the `migration_report.csv` generated by Epic 2. For every audiobook flagged as `[PENDING UPLOAD]`, it will execute a non-destructive `Copy-Item` to mirror the folder to a designated external cloud sync drive. It must entirely ignore `[SYNCED]` items.

## Proposed Changes
### `[NEW]` `tests/Migrate-Audiobooks.ps1`
- **Report Ingestion:** Read the `migration_report.csv`.
- **Target Filtering:** Isolate only rows where `Status -eq "[PENDING UPLOAD]"`.
- **Execution Engine:** Use `Copy-Item -Recurse` to clone the folder from the source directory to the destination directory.
- **Defensive Coding:** `Try/Catch` blocks for file I/O. `-LiteralPath` strictly enforced. Ensure source files are never deleted.

## Verification Plan
1. The Gatekeeper will audit the code against the `ALLOW/DENY` protocol syntax to verify no destructive operations (`Remove-Item`) exist.
2. The Developer will write `tests/test_migration.ps1` to generate a mock CSV and mock folders, proving the `Copy-Item` routing is mathematically accurate before merging.
